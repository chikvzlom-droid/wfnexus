1. ПРОЕКТ
================================================================================

Warframe Nexus — десктопный ассистент для анализа рынка Warframe,
управления инвентарём и автоматизации торговли.

Стек:
- Backend: FastAPI (Python 3.14), SQLAlchemy async, SQLite, numpy
- Frontend: React + TypeScript, Vite, Tailwind CSS v4, React Router
- База: SQLite (файл warframe_nexus.db)

4. КЛЮЧЕВЫЕ РЕШЕНИЯ И КОНСТАНТЫ
================================================================================

WFM API:
- Cookie auth (JWT=<token>), не Authorization header
- /me возвращает data.payload.profile или data.profile или data.user
- Slug пользователя: skanixxx
- Rate limit: 2 req/s, burst 5 (RateLimiter в client.py)
- /orders/user/{slug} — публичный, без аутентификации

Синхронизация:
- Полный синк только при старте если БД пуста
- Полный синк: каждые 6 часов
- Watchlist синк: каждые 30 минут
- Проверка цен: каждые 5 минут
- SMA(12): среднее closed_price за последние 12 снапшотов

Торговля:
- Импорт: создаёт TradeOrder для каждого WFM ордера без локального wfm_id
- Авто-завершение: если wfm_id исчез из WFM → status=completed + Transaction
- Profit: от последней покупки (FIFO-like)

Прочее:
- subcategory на Item всегда None (WFM API не отдаёт)
- ducats парсятся из raw.get("ducats") — есть только на прайм/синдикат шмотках
- Иконка в открытом конфиге утекла? не критично, но напомнить если надо
